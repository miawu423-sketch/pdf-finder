PDF Finder

安装后在论文页面点击书签即可使用 | 自动获取最新版本

备用安装方式

  1. 右键书签栏 → 添加书签
  2. 名称填 PDF Finder
  3. 网址栏粘贴下面的代码

功能特点

自动更新

脚本托管在云端，每次点击加载最新版本

智能识别

自动区分本文 PDF 和推荐文章链接

付费提醒

检测到付费文章时显示警告

一键复制

只有一条结果时自动复制到剪贴板

支持的学术网站

Springer / Nature / Elsevier / Wiley / ACM / IEEE* / MDPI / arXiv / Frontiers / Taylor & Francis / SAGE 等主流出版商

* IEEE 需先登录机构账号
